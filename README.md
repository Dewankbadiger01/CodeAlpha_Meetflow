# 🎥 MeetFlow

MeetFlow is a modern video meeting web application built using the MERN stack. It provides a clean and responsive interface for creating, joining, and managing online meetings. The project focuses on an intuitive user experience with a professional meeting dashboard and meeting room.

---

## 🚀 Features

* 🔐 User Authentication (Login & Register)
* 📊 Dashboard with meeting overview
* 📅 Upcoming & Recent Meetings
* ➕ Create New Meetings
* 🔗 Join Meetings using Meeting ID
* 🎥 Responsive Meeting Room UI
* 🎙️ Microphone & Camera Controls
* 💬 Chat Sidebar
* 👥 Participant Panel
* ⚙️ Meeting Settings
* 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

### Frontend

* React.js

* React Router DOM
* Axios
* React Icons

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcrypt.js

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/MeetFlow.git
cd MeetFlow
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```
